# Contributing to Interactive Portfolio

Thank you for your interest in contributing to this project! Your help is greatly appreciated. Please follow the guidelines below to ensure a smooth contribution process.

## Getting Started

1. **Fork the repository** and clone it to your local machine.
2. **Install dependencies** using `npm install`.
3. **Create a new branch** for your feature or bugfix: `git checkout -b feature/your-feature-name`.

## Development

- Use `npm run dev` to start the development server.
- Make sure your code follows the existing style and conventions.
- Write clear, concise commit messages.
- Test your changes across different screen sizes for responsiveness.

## Pull Requests

- Ensure your branch is up to date with `main` before submitting a pull request.
- Provide a clear description of your changes and the motivation behind them.
- Reference any related issues in your pull request description.
- Be responsive to feedback and make requested changes promptly.

## Code Style

- Use React (TypeScript) and functional components.
- Prefer clean, minimal, and accessible UI.
- Use CSS variables and Tailwind CSS for styling.
- Keep components modular and reusable.

## Issues

- If you find a bug or have a feature request, please open an issue with detailed information.

## Code of Conduct

- Be respectful and inclusive.
- No harassment, discrimination, or inappropriate behavior will be tolerated.

Thank you for helping make this project better!
