# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability, please email mwakidenice@gmail.com with details. I will respond as soon as possible.

## Supported Versions
Only the latest deployed version is supported for security updates.

## Security Best Practices
- Do not share sensitive information in public issues or pull requests.
- Use strong, unique passwords for all accounts.
- Keep dependencies up to date.
- Review and test all code changes for security risks before deployment.

## Responsible Disclosure
I appreciate responsible disclosure of security issues. Please give ME reasonable time to address the issue before public disclosure.

## Contact
For any security concerns, contact: mwakidenice@gmail.com
